10BucksClub

10BucksClub is a badminton tournament helper app designed to simplify team assignments, session management, and match scheduling for tournament organizers. It enhances efficiency and reduces administrative overhead, allowing organizers to focus on running successful events.

Key Features

  • Automated team assignments based on participant lists
  • Session management, including tracking attendees and waitlists
  • Match fixture creation and real-time score calculation
  • Streamlined tournament flow with minimal admin effort

Technical Highlights

  • SwiftUI for a modern and responsive user interface
  • MVVM Architecture for maintainability and scalability
  • Core Data for efficient local data storage and retrieval
  • Custom algorithms for team assignment and score computation

MyFinanceTracker

A personal finance app that empowers users to track expenses, manage budgets, and visualize spending habits.

Key Features

  • Real-Time Expense Tracking
  • Budget Management and Alerts
  • Data Visualization with Charts & Graphs
  • Secure Storage with Core Data and iCloud Sync

Technical Highlights

  • SwiftUI for a responsive and modern UI
  • MVVM Architecture for maintainable code
  • Core Data for persistent, offline storage
  • Integration with iCloud for syncing across devices

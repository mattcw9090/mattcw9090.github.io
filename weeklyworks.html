WeeklyWorks

A weekly scheduler that manages students' training sessions for the current week. It automates venue bookings, sends messages, and adds events to Google Calendar. This software is ideal for any badminton coach who needs to manage their coaching schedule on a weekly basis efficiently.

Key Features

  • Weekly Training Session Management
  • Automated Venue Booking System
  • Automated Messaging and Notifications to Students
  • Integration with Google Calendar for Schedule Syncing
  • User-Friendly Dashboard for Coaches

Technical Highlights

  • SwiftUI for a modern and responsive user interface
  • MVVM Architecture for maintainable and scalable code structure
  • Core Data for efficient local data storage and retrieval
  • API Integration for Google Calendar Syncing
  • Automated Messaging via SMS or Email APIs

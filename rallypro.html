RallyPro

A tournament software match maker that takes the current players in a session and generates matches. The software also keeps track of who should be attending, who should be on the waitlist, and maintains match results and individual stats. It is an excellent tool for any badminton tournament admin or organizer.

Key Features

  • Automated Match Generation Based on Current Players
  • Attendance Tracking and Waitlist Management
  • Real-Time Match Results and Score Tracking
  • Individual Player Statistics and Performance Analytics
  • User-Friendly Interface for Tournament Administrators

Technical Highlights

  • SwiftUI for a modern and responsive user interface
  • MVVM Architecture for maintainable and scalable code structure
  • Core Data for efficient local data storage and retrieval
  • Custom Algorithms for Match Generation and Statistics Computation
  • Local Notifications for Real-Time Updates
